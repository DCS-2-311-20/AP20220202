    ![ループ](ap20220202b.png)


    * ap0202c.html で，図のように 20個の乱数の合計，平均，標準偏差が表示されるように，ap0202c.js を修正する．

    ![合計](ap20220202c.png)

    * ap0202d.html で，図のような出力が表示されるように，ap0202d.js を修正する．
